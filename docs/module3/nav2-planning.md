# Nav2 for Humanoids

## Navigation Stack
[cite_start]**Nav2** is used for advanced path planning, specifically tailored here for **bipedal humanoid movement**[cite: 67].

## Functionality
* **Path Planning:** Calculates optimal paths while avoiding obstacles.
* [cite_start]**Navigation:** Controls the robot's movement to follow the planned path within the NVIDIA Isaac ecosystem[cite: 66, 67].
